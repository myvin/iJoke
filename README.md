# ijoke

Command-line joke generator

# Installation



# Usage

* `ijoke`

Generate a joke.

* `ijoke -c 5` or `ijoke --count 5`

Generate 5 jokes.

* All options:

```bash
-c, --count     Set output joke count
-h, --help      Print ijoke command line options
-v, --version   Print current ijoke version
```
